Orm

This feature setups NHibernate which registers IEntityContext and IQueryContext implementations and adds AutoPersistenceModel and AutoMapping conventions along with an Instantiator interceptor for NHibernate.

Add this feature using AddOrm() extension;

app.Features.AddOrm(...);
