Database

This feature provides a database setup to be used both in development and production setups.

Add this feature using AddDatabase() extension;

app.Features.AddDatabase(...);

In Memory

Adds in-memory database setup.

c => c.InMemory()

Sqlite

Adds local Sqlite database setup which creates local sqlite database in documents folder with given name.

c => c.Sqlite(fileName: "test.db", autoExportSchema: false)

MySql

Adds MySql database setup which gets connection parameters from app.settings.

c => c.MySql()
