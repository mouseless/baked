Testing

DO uses NUnit for unit testing, Moq for mocking and Shouldly for assertion.

app.Layers.AddTesting();

Configuration Targets

Testing layer only exposes TestConfiguration for configuration target.

TestConfiguration

This target is provided in AddServices phase. To configure it in a feature;

configurator.ConfigureTestConfiguration(test =>
{
    ...
});

Phases

This layer introduces following phases to the application it is added;

  • CreateConfigurationManager: This phase runs as the earliest stage of a test run to add an empty ConfigurationManager to the application context.
  • Run: This phase is added internally and is the latest phase in a test run. It is not allowed to provide a configuration at this phase, it only builds given service collection and adds IServiceProvider to the application context to be used by specs.
