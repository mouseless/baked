Data Source

Data source recipe includes enough layers and feature implementations for a backend application that is expected to provide data from configured data source using report queries in .sql files.

To create an application from this recipe, use DataSource() extension of Bake class directly in Program.cs.

Bake.New
    .DataSource(
        business: c => c.DomainAssemblies([...])
    )
    .Run();
