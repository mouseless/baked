Release Notes

Releases follow [major].[minor].[patch] Semantic Versioning format;

  • [major]: This part will only increase with the major .NET releases with no backward compatibility.
  • [minor]: This will increase when there is at least one new feature or breaking change.
  • [patch]: For release that has only a bugfix or an improvement, patch number will increase.
