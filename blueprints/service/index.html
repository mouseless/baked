Service

Service blueprint is the default backend blueprint provided by DO which includes necessary layers and feature implementations for any backend application.

This blueprint is under development and will be detailed as it becomes more mature.

To create an application from this blueprint, use Service() extension of Forge class directly in Program.cs.

Forge.New
    .Service(
        business: c => c.MyBusiness(),
        database: c => c.Sqlite()
    )
    .Run();
